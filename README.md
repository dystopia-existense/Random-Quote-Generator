# Random Quote Generator

**Автор:** Вакулин Иван

## Описание
GUI-приложение на Python (tkinter) для случайной генерации цитат с сохранением истории в JSON. Поддерживает фильтрацию по автору и теме, добавление пользовательских цитат, проверку пустого ввода.

## Примеры использования
- **Генерация:** нажать «Сгенерировать цитату» – случайная цитата отобразится в поле и запишется в историю.
- **История:** все сгенерированные цитаты отображаются в прокручиваемом списке.
- **Фильтрация:** выбрать автора/тему из выпадающих списков – в истории остаются только соответствующие записи. Кнопка «Сбросить фильтр» возвращает полный список.
- **Добавление цитаты:** ввести текст (обязательно), автора, тему → «Добавить цитату в пул». Цитата станет доступной для генерации.
- **Сохранение/загрузка:** автоматические при закрытии/открытии, доступны ручные кнопки «Сохранить историю» и «Загрузить историю заново».
